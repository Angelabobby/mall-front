vue-cli创建项目

npm i less less-loader@5 --save

npm i vue-router --save



```json
// package.json
"scripts": {
    "serve": "vue-cli-service serve --open",
    "build": "vue-cli-service build",
    "lint": "vue-cli-service lint"
},
```

应该使用 --open --host localhost，指定host地址，否则host默认为http://0.0.0.0/
